# 💳 ATM Interface - Java Swing + MySQL

A modern desktop-based **ATM simulation system** built with **Java Swing** for the frontend and **MySQL** for the backend. This project is created as part of my **Java Development Internship at Oasis Infobyte**, showcasing core functionalities of an ATM system in a secure and user-friendly interface.

---

## 🚀 Features

🔐 **Login System**  
- Secure login using credentials stored in MySQL database

🏦 **ATM Dashboard**  
Once logged in, users can perform the following operations:

- 💰 **Check Balance**  
  - Displays the current balance of the user’s account

- 💵 **Deposit Money**  
  - Adds amount to the account and updates the balance

- 🧾 **Withdraw Money**  
  - Deducts amount from balance after verifying sufficient funds

- 🔁 **Transfer Funds**  
  - Allows user to transfer money to another account securely

- 📜 **Transaction History**  
  - Shows past transactions including deposits, withdrawals, and transfers

- 🔓 **Logout**  
  - Ends the session and returns to the login screen
