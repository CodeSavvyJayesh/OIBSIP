# 🎮 Number Guessing Game (Java Swing GUI)

A fun and interactive number guessing game built using **Java Swing**, featuring hints, scoring system, dark mode toggle, restart functionality, and a beautiful GUI.

---

## 🧠 Features

- 🔢 Random number generation between 1 to 100
- 💡 3-step Hint system (Even/Odd, Greater than 50, Divisible by 5)
- 🌗 Dark Mode toggle
- 🎯 Score calculation based on attempts
- 🔁 Restart functionality
- 🎨 Styled buttons and fields using Java Swing
- 🚫 Limited attempts (5 per round)

---

## 💻 Technologies Used

- Java (JDK 8 or above)
- Swing (Java GUI library)
- IntelliJ IDEA (recommended)

## 🚀 How to Run the Project from IntelliJ Terminal

Follow the steps below in your IntelliJ terminal:

```bash
cd NumberGuessingGame-main
cd NumberGuessingGame
cd src
javac NumberGuessingGame.java
java NumberGuessingGame


🧾 Game Rules
1.You have 5 attempts to guess the number.

2.Click on HINT (max 3 times) for clues.

3.Use the Dark Mode checkbox to toggle theme.

4.Score = (Attempts Left + 1) * 10.

5.Click Restart to play a new round after winning or losing.
